📊 European Pharmacy Sales Analysis – Power BI Project

📌 Project Overview

This project presents an end-to-end Exploratory Data Analysis (EDA) and interactive dashboard built using Power BI to analyze pharmacy sales performance across Europe.

The goal of this analysis is to uncover key business insights related to sales, profit, regional performance, and pharmacy-level efficiency.

🎯 Objectives
Analyze overall sales and profitability trends
Identify top-performing and underperforming regions
Evaluate category-wise sales contribution
Detect at-risk pharmacies
Understand year-over-year growth patterns
📈 Key Metrics
Total Sales: €8.63M+
Total Profit: €2.42M
Profit Margin: 28.04%
Total Units Sold: 446K
Total Pharmacies: 120
🔍 Key Insights
Prescription category contributes the highest share of sales
Germany, France, and Italy are leading in revenue and margin
Some regions show high sales but lower margins, indicating optimization opportunities
Positive correlation between margin and units sold across cities
Stable YoY growth (~4%), reflecting consistent performance
Identified a few at-risk pharmacies requiring attention
📊 Dashboard Features
Executive Summary with KPIs
Category-wise and Region-wise analysis
Monthly sales & units trend
Geographical performance (city & country level)
Pharmacy-level performance tracking
Store segmentation (Urban, Suburban, Rural)
🛠 Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Data Modeling
